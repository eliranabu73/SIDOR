GreedyProvider או OR-Tools.

weights: availability, fairness, weeklyHours...

←
✅
אישור הצעות

POST /v1/schedules/:id/apply-proposals POST /v1/schedules/:id/publish
מצבי משמרת (Shift States)
⬜ OPEN — פתוח 🟦 PLANNED — משובץ (draft) 🟩 ASSIGNED — מאושר 🔒 LOCKED — נעול 🔴 CONFLICT — התנגשות 🔵 PUBLISHED — פורסם לעובדים
➕
יצירת משמרת ידנית
scheduleId, locationId, roleId, startAtUtc, endAtUtc
POST /v1/shifts → PATCH /v1/shifts/:id → DELETE /v1/shifts/:id

📡 כל ה-API Endpoints — מיפוי מלא

GET /v1/me פרופיל + memberships
POST /v1/orgs יצירת ארגון + membership + location + schedule
GET /v1/templates רשימת 13 תבניות
POST /v1/templates/:id/apply החלת תבנית — upsert roles + shifts
GET /v1/schedules/ensure קבל/צור schedule לפי weekStart
POST /v1/schedules/:id/auto-schedule הרצת scheduler (greedy/or-tools, dryRun)
POST /v1/schedules/:id/apply-proposals אישור הצעות שיבוץ
POST /v1/schedules/:id/publish פרסום לוח → PUBLISHED status
POST /v1/shifts יצירת משמרת ידנית
PATCH /v1/shifts/:id עדכון שעות/תפקיד/סניף
DELETE /v1/shifts/:id ביטול משמרת
POST /v1/employees הוספת עובד
PATCH /v1/employees/:id עדכון עובד (roles, location, phone...)
DELETE /v1/employees/:id מחיקה רכה (soft delete)
POST /v1/roles יצירת תפקיד חדש
POST /v1/locations יצירת סניף חדש
POST /v1/swaps בקשת החלפת משמרת
GET /v1/schedules/:id/fairness מדדי הוגנות
GET /v1/schedules/:id/labor-cost עלות עבודה
GET /v1/realtime/schedules/:id SSE stream — עדכונים חיים
PATCH /v1/settings עדכון הגדרות ארגון
POST /v1/import ייבוא CSV עובדים
